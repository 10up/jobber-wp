# Jobber Plugin E2E Tests

This directory contains end-to-end tests for the Jobber WordPress plugin.

## Directory Structure

- `specs/` - Test specification files
- `config/` - Configuration files
- `dummy-data/` - Test fixture data
- `test-results/` - Test execution results and reports

## Running Tests

To run the tests:

```bash
npm run test:e2e
```

## Configuration

Test configuration can be found in:
- `playwright.config.js` - Playwright configuration
- `config/` - Test-specific configuration
